SwiftSyncLITE - User Manual


App Overview

SwiftSyncLITE is a lightweight synchronization tool designed to keep your files up-to-date between two directories. It provides
quick and efficient synchronization features for managing your data.

When you run the synchronize command, your master
and local directory will begin syncing. Any differences that the app notices between your local and master file directories,
SwiftSyncLITE will then resolve all the differences.

Profiles

Profiles store your data for that instance of the directories you're syncing, the mode you're syncing with, and other settings.
This way, when you want to switch between your sync instances, you can simply go to the Profiles tab and load one from a file.

To create a new profile, follow these steps:

  1. Go to Profiles > Manage Profiles.
  2. Click on 'Create New.'
  3. Follow the on-screen instructions to set up your master and local directories.
  4. Save the profile to a file location of choice for future use. If you do not save your profile after creating it, the data will be lost upon closing the app.

Commands

CommandDescription
sync (or synchronize)Start synchronization. Make sure to load a profile first.
master rootOpen the master directory in the file browser.
local rootOpen the local directory in the file browser.
master sizeShow the size of the master directory.
local sizeShow the size of the local directory.
status hide (or hide status)Hide the status/process bar.
status show (or show status)Show the status/process bar.
helpShow this help menu.
profile pageSwitch to the profile management page.
